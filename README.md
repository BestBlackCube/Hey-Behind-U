# Hey-Behind-U
## 게임 소개
[outLast](https://store.steampowered.com/app/238320/Outlast/)의 1인칭 추격자에게서 도망가는 시스템과, [REPO](https://store.steampowered.com/app/3241660/REPO/), [Lethal Company](https://store.steampowered.com/app/1966720/Lethal_Company/)에서 나오는
추격자를 몇몇의 시스템을  
벤치마킹 하여 기능을 개발한 에드온입니다.

## 프로젝트 소개
저희 기업 연계 프로젝트2 23조의 주제는 "3D 게임 기능 개발"으로 선정하여 팀원들의 역할 재배치와  
기업 연계 프로젝트1 에선 못해봤던 일들을 해보며, 엔진에 대한 이해도 학습을 우선시 했습니다.

언리얼 엔진으로 3D 런어웨이 게임의 기능을 구현하자 라는 목표로 만들려던 게임이였지만,  
팀원들의 엔진 미숙지와 C++의 능력 부족으로 유니티 엔진으로 바꾸어 만든 게임 에드온입니다.

## 프로젝트 기획
추격자에게서 도망가는 게임이다 보니 추격자가 플레이어를 따라가는 반 Ai 시스템 구현과  
3D 모델링, 대형 맵 등 분위기와 맞는 에셋을 찾아 모델리의 이해도와 소스 코드 개발 능력을 늘리는 것입니다.  

### 팀원 역할
- ### 기획: 레퍼런스 분석 및 레벨 디자인, 게임 시스템 설계
    - 다양한 추격자의 탐색, 추격 시스템을 구상하고, 전체적인 게임 흐름을 기획합니다.
- ### 게임 로직: 게임 시스템 구현, 클라이언트 로직 개발
    - 추격자의 탐색, 추격의 반 Ai 시스템을 개발하고 플레이어 캐릭터와 상호작용하는  
    기능들을 개발합니다.
- ### 오디오: 오디오 소스 제작 및 편집
    - CC0 (Creative Commons Zero)를 사용하여 전반적인 오디오 소스를 확보하여 원본 소스는  
    그대로 사용하지만, 필요한 소스는 편집 프로그램을 사용하여 제작합니다.
- ### GUI: 이미지, 에셋, UI 소스 제작 및 편집
    - 일러스트, 배경 같은 이미지는 스테이블 디퓨전 AI를 사용하여 이미지를 제작하고, 캐릭터  
    같은 경우 유니티 에셋 또는 itch.io 등을 사용하여 무료, 유로 버전의 소스를 사용한다.
